Class: SliderControl

SliderControl

new SliderControl(home, id, options)

Construct a new Slider control.
Parameters:
Name Type Description
home Home the home that this control belongs to.
id String the id of this control. Updates will be POSTed to home.control_endpoint/id
options Object options for this control:
  • name - String, required - friendly name for this control
  • min - float, required - minimum value.
  • max - float, required - maximum value.
  • value - float, required - initial value.
  • step - float, required - amount of one slider step.
